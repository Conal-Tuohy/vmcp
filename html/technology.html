The software was developed for the Gardens by Conal Tuohy, a freelance software developer based in Brisbane. There are two parts to the system:

  • a pipeline which converts more than 10,000 items of Mueller's correspondence from Microsoft Word format into TEI XML documents,
  • a web application which publishes the resulting TEI XML files as a searchable website.

The high level structure of the web application is a pipeline written in the XProc 1.0 scripting language. Within the XProc pipeline, the TEI XML files and HTML pages are processed using XSLT 3.0 stylesheets. The XProc pipeline is hosted by a Java web Servlet called XProc-Z, which in turn is hosted in the Java web server Apache Tomcat.

The web application uses the Apache Solr to as a search engine to drive the search and browse interface, and perform hit-highlighting.
